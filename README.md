## 🎮 조작법
| 키 입력    | 동작    |
|---------|-------|
| `W`     | 위로 이동 |
| `A`     | 왼쪽 이동 |
| `S`     | 아래로 이동 |
| `D`     | 오른쪽 이동 |
| `E`     | 상호작용  |
| `Shift` | 달리기   |
| `Space` | 점프    |
| `Tab`   | 인벤토리  |
